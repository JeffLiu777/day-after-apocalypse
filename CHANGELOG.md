## 更新日志 / Changelog

### beta0.25

当前版本主要围绕 **上线试玩、界面观感、宠物系统和基础交互体验** 进行调整。

#### 主要更新

- 游戏已部署为网页试玩版，可通过 GitHub Pages 在线游玩。
- 修复开局黑屏 / 闪屏问题，打开网页后会直接进入主菜单。
- 主菜单流程调整为：
  - 先显示「新游戏 / 读档」
  - 点击「新游戏」后再进入姓名、职业、特质选择
- 优化顶部功能按钮，将大量按钮按功能分组：
  - 常用
  - 探索
  - 家园
  - 角色
  - 系统
  - 全部
- 重做宠物互动界面，提高文字亮度、背景对比度和弹窗层级。
- 宠物互动结果现在会显示在宠物面板内，不再只刷到左下角日志。
- 恢复宠物具体品种选择，不再只能选择「猫」或「狗」。
- 扩展宠物互动：
  - 呼唤
  - 抚摸
  - 陪玩
  - 检查状态
  - 狗狗巡逻
  - 猫猫自由活动
- 宠物可喂食物扩展：
  - 狗粮
  - 猫粮
  - 肉类罐头
  - 罐头鱼
  - 午餐肉
  - 火腿肠
  - 火腿
  - 牛肉干
  - 肉干
- 提升左下角日志文字可读性，增强颜色、亮度和阴影。
- 修复开局提示弹窗反复出现的问题。
- 新增开局提示文本：
  > 我为了寻找我的家人不得不离开这个避难所......如果还有人来到这里，一定要多收集材料，升级扩张你的避难所，招募更多的伙伴,每位伙伴在不同的好感度阶段会有不同的剧情故事和对话。祝你好运。 ——上一任避难所拥有者

#### NPC 与对话

- 修复 NPC 招募问题提前出现的问题。
- 所有 NPC 的特殊招募问题现在只会在点击「邀请加入避难所」后触发。
- 已加入避难所的 NPC 不再显示「邀请加入避难所」。
- 室外再次遇到已加入 NPC 时，会直接进入正式互动。
- 选项对话改为独立触发，不会一打开 NPC 面板就弹出问题。
- 每个选项问题只能触发一次。
- 答对选项问题会提升好感度，答错会降低好感度。
- 林小果在避难所内新增每日治疗功能，每天可恢复最多 50 点生命值。

#### 系统与玩法

- 新增或强化以下玩法原型：
  - NPC 每日消耗食物和水
  - 仓库不足时按好感度处理 NPC 状态惩罚
  - 武器耐久度
  - 季节影响
  - 种植系统
  - 烹饪系统
  - 角色任务雏形
  - 车辆远征雏形
  - 每日事件卡片
  - NPC 互相关系雏形
  - 噪音值系统
  - 侦察 / 偷袭 / 战斗流程雏形
- 调整背包物品使用逻辑，支持指定吃什么、喝什么、使用什么。
- 背包类物品可以装备，并提升负重上限。
- 搜索 UI 改为更大的检索窗口：
  - 普通物资更快出现
  - 高级物资需要更长搜索时间
- 删除部分场景内不必要的 emoji / ASCII 风格元素，使画面更依赖真实场景图。

#### 美术与场景

- 已内嵌大量像素风场景图，单文件打开也能显示主要场景。
- 已内嵌 NPC 头像，避免本地打开时出现图片丢失。
- 场景背景会根据当前地点切换。
- 避难所、居民区、医院、警局、仓库、药店、五金店、农场、地铁入口等地点已拥有对应背景图。

#### 已知问题

- 当前版本仍处于早期 Demo 阶段，部分系统还只是原型。
- UI 仍然存在信息量较大的问题，后续需要继续精简按钮和菜单层级。
- 部分玩法数值尚未平衡。
- 战斗、种植、烹饪、车辆远征等系统仍需要进一步打磨。
- 旧版本存档在未来大改后可能出现兼容问题。
