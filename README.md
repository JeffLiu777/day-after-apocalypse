# 末日余生 / Day after Apocalypse

**《末日余生》** 是一款以末日生存、避难所经营、角色互动和文字叙事为核心的网页游戏。  
玩家将在灾变后的废弃避难所中醒来，面对食物短缺、水源紧张、丧尸威胁、幸存者关系和一次次艰难选择。

这是一个仍在开发中的个人独立游戏 Demo。  
目前游戏还不完美，系统、数值、UI 和剧情都在持续调整中。如果你愿意试玩并留下意见，我会非常感谢。

## 游戏简介

在《末日余生》中，你不是一开始就拥有强大基地的主角。  
你只是一个在末日中醒来的普通幸存者。

你需要在城市废墟、居民楼、医院、警察局、仓库、超市、农场等地点中探索，寻找食物、水、药品、工具、武器和材料。  
你需要管理自己的生命、饥饿、口渴、心情和体力，也要考虑避难所是否足够安全、资源是否够用、伙伴是否信任你。

随着时间推进，你会遇到不同的幸存者。  
他们不是简单的工具人，而是有性格、有过去、有喜好、有底线的人。  
你可以尝试招募他们，和他们聊天，送礼物，提升好感度，逐渐解锁他们的故事。

有的人会保护你，有的人会治疗你，有的人能修理设备，有的人擅长侦察，也有人可能因为不信任你而离开。

在这个世界里，活下去并不只是“搜刮更多物资”。  
有时候，如何对待身边的人，才是更难的选择。

## 幸存者角色

目前游戏中已有多位可遇见或可招募的幸存者：

- **林小果**：医生，理性、克制，但并不冷漠
- **谷雨**：植物学家，喜欢周杰伦、NANA 和猫
- **周海山**：加油站工人，嗜烟如命，嘴硬心软
- **董澈**：侦察员，行动轻快，熟悉路线和观察
- **韩队**：神秘守夜人，沉默可靠，习惯守夜
- **秦亦可**：电气工程师，擅长设备、电路和维修
- **马库斯**：运动员，身材高大，热爱篮球和音乐

每位角色都有自己的背景、喜好、招募条件、随机对话和好感度剧情。  
随着好感度提升，他们会逐渐说出更多过去，也可能和玩家形成更深的关系。

## 宠物伙伴

在探索过程中，玩家可能会遇到动物伙伴。  
宠物不只是“消耗食物”的存在，它们也会对玩家产生反应。

例如：

- 狗会摇尾巴、蹭你、听见呼唤跑过来，也可以巡逻
- 猫会假装没听见、蹭手、跳上柜子、偶尔叼回小东西
- 宠物可以通过喂食、陪玩、抚摸等方式提升亲密度
- 不同宠物会有不同的性格和互动反馈

目前宠物系统还在继续扩展中。

## 存档说明

本游戏目前使用浏览器本地存档。

这意味着：

- 你的存档只保存在你自己的浏览器中
- 你玩游戏不会影响其他玩家的存档
- 其他玩家也不会影响你的存档
- 网站更新通常不会删除你的存档
- 如果清理浏览器缓存 / 网站数据，可能会导致存档丢失

如果后续版本进行大幅系统更新，旧存档可能出现不兼容情况。  
建议重要版本更新前手动备份存档。
